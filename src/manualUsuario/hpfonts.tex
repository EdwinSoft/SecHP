\section*{Installation and Usage}

Install the HP Prime font using the {\tt HPPrime.ttf} file.
Then use the following template and table of macros below
to create your document.

\begin{quote}
\begin{verbatim}
\documentclass{article}
\usepackage{fontspec,hpprime}

\begin{document}
   ...
\end{document}
\end{verbatim}
\end{quote}
